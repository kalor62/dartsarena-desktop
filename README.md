# DartsArena Desktop

Aplikacja desktopowa do gry DartsArena — łączy lokalny Autodarts Board Manager z [DartsArena](https://dartsarena.pl).

## Pobieranie

Najnowsza wersja: **[Releases](https://github.com/kalor62/dartsarena-desktop/releases/latest)**

| Platforma | Plik |
|-----------|------|
| macOS (Apple Silicon) | `DartsArena-Desktop-<v>-arm64.dmg` |
| macOS (Intel) | `DartsArena-Desktop-<v>.dmg` |
| Windows | `dartsarena-desktop-<v>-setup.exe` (installer) lub `DartsArena-Desktop-<v>.exe` (portable) |
| Linux | `DartsArena-Desktop-<v>.AppImage` |

## Auto-update

Aplikacja sprawdza i pobiera aktualizacje w tle. Wystarczy ją zrestartować, gdy nowa wersja zostanie pobrana.

## Source

Kod źródłowy: prywatne repo `dartsarena.pl`. Repo to hostuje wyłącznie podpisane binarki + manifesty auto-updatera.
